# 招标文件编制

基于 Figma 设计稿实现的招标文件编制工作台。

## 页面模块

- **蓝色顶栏** — 项目标题、保存/版本/导出、用户信息
- **工具栏** — 文件上传 / 项目信息 / 范本选择 三个 Tab
- **文件上传** — 立项批文、工程量清单、图纸、其他文件上传，生成招标文件
- **项目信息** — 项目基本信息与招标人信息表单
- **范本选择** — 智能推荐范本 + 范本库搜索选择
- **文档预览区** — 加载测试 PDF 招标文件，支持滚动阅读
- **文档目录** — 左上角图标展开/收起 PDF 书签目录，点击跳转对应页

测试 PDF 位于 `public/documents/sample-bidding-doc.pdf`（黄州区档案馆项目招标文件，117 页）。

## 启动

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5174
